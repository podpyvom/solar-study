# Solar Study Telegram Mini App

Учебное мини-приложение для Telegram реализовано на Node.js. Оно запускает Express веб‑сервер и бота Telegraf, открывающего мини‑приложение по команде `/start`.

## Запуск

1. Укажите токен бота в файле `config.js` вместо `YOUR_TELEGRAM_TOKEN`.
2. Выполните одну команду:

```bash
./start.sh
```

Скрипт установит зависимости и запустит сервер на `8080` вместе с ботом.

## Настройка BotFather

1. Создайте бота через [BotFather](https://t.me/BotFather) и получите токен.
2. В настройках `Web App` укажите ссылку на ваш сервер (`https://<domain>:8080/`).
3. Поместите токен в `config.js`.

## Структура

- `src/` — исходный код сервера и бота.
- `views/` — HTML страницы мини‑приложения.
- `public/` — статические файлы (CSS и JS).
- `data/` — загруженные файлы и метаданные.
- `docs/` — тексты пользовательского соглашения и политики.
- `start.sh` — скрипт запуска.

Файлы сохраняются в каталоге `data/uploads` под именем `университет_преподаватель_курс_название_время.ext`. Информация о файлах хранится в `data/meta.json` и используется при поиске.
